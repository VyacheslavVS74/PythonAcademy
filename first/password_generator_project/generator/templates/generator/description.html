Описание генерации паролей

Для генерации пароля выберите кол-во символов. Также если необходимо, есть возможность добавить в пароль - верхний регистр, цифры и спецсимволы.

Далее нажмите "Сгенерировать пароль".

Если хотите обновить пароль с действующими настройками нажмите кнопку "Обновить пароль". Чтобы изменить настройки генерации пароля нажмите кнопку "Вернуться на главную"
